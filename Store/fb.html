Facebook API CORS Bypass

Sử dụng corsproxy.io để gửi yêu cầu từ trình duyệt mà không bị chặn.

Chưa có dữ liệu...

Lưu ý bảo mật:

  • Không bao giờ để lộ App Secret hoặc Access Token trong code client-side khi chạy thực tế (Production).
  • Proxy miễn phí có thể lưu nhật ký (logs), không dùng cho dữ liệu cực kỳ nhạy cảm.
  • Cách tốt nhất là tạo một Backend nhỏ (Node.js, Python) để gọi API từ Server-to-Server.
