Hướng dẫn cài đặt Windows

Bước 1/8
1

Tải bộ cài Chesino Auto Windows

Bộ cài giúp bạn cài Windows và phần mềm nhanh chóng. Hiện tại có 2 phiên bản:

  • Basic: Chỉ các phần mềm cơ bản [380Mb]
  • Plus: Phần mềm boot Anhdv, Windows 10 & 11 và các phần mềm cơ bản [2.7Gb]
2

Tạo USB Boot

Hãy làm theo hướng dẫn trong video dưới đây để tạo USB Boot:

3

Kiểm tra phím tắt Menu Boot & BIOS

Chạy file "Kiểm tra phím vào BIOS, Menu BOOT" trong bộ công cụ.

Cửa sổ hiển thị thông tin phím tắt:

Hãy nhớ sao lưu dữ liệu quan trọng ra thiết bị khác trước khi tiến hành cài đặt!

4

Vào USB Boot

Khởi động lại máy và nhấn liên tục phím tắt Menu Boot (Ví dụ ASUS là F8).

Trong danh sách hiện ra, chọn dòng có tên USB của bạn (thường có chữ UEFI).

5

Phân vùng ổ cứng

Khi vào WinPE, mở phần mềm "MiniTool Partition Wizard".

Cẩn thận chọn đúng ổ cứng cần cài Win. Dữ liệu sẽ bị xóa sạch!

5.1 Xóa phân vùng cũ

Chuột phải vào ổ cứng > Chọn Delete All Partitions.

5.2 Tạo phân vùng EFI & Windows

Chuột phải vùng trống (Unallocated) > Create.

  • EFI: Size 100-300MB, File System: FAT32.
  • Windows: Dung lượng còn lại, File System: NTFS.
6

Cài đặt Windows

Mở file ISO Windows (Mount ra ổ ảo) hoặc vào thư mục chứa bộ cài.

Chạy file Setup.exe.

  • Bấm Next > Install Now.
  • Chọn "I don't have a product key" nếu được hỏi.
  • Chọn phiên bản Windows 10/11 Pro.
  • Chọn phân vùng đã đặt tên là "Windows" ở bước trước.
7

Cập nhật Driver (Update Windows)

Sau khi vào Win, vào Settings > Update & Security.

Bấm Check for updates và đợi máy tự tải Driver. Restart khi được yêu cầu.

8

Kích hoạt Windows & Office

Lưu ý: Sử dụng công cụ kích hoạt (Active) có thể bị trình diệt virus chặn. Hãy tắt Real-time protection trước.

Sử dụng công cụ Microsoft Toolkit có sẵn trong bộ cài.

Chúc mừng! Bạn đã cài đặt xong Windows.
