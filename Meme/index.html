Từ & Nghĩa Thao tác

Tải lên Hàng Loạt File

Chọn nhiều ảnh/video cùng lúc. Hệ thống sẽ tự động tải lên và đặt tên tự động theo số thứ tự (Ví dụ: Meme A-0001). Tags mặc định sẽ để trống.

Nhập Dữ Liệu Nâng Cao

Copy từ Excel (URL | Tiêu đề | Tags) và ấn Ctrl+V vào ô Nguồn Media.
Dán mảng JSON. Định dạng: [{"url": "...", "title": "...", "tags": ["tag1", "tag2"]}]
Nhập file .json để tải lên, hoặc tải file Backup về máy.

Nhập Dữ Liệu Nâng Cao

Copy từ Excel (Từ | Nghĩa | Ví dụ | Tags) và dán vào ô Từ khoá.
Dán mảng JSON. Định dạng: [{"term": "...", "definition": "...", "example": "...", "tags": ["a"]}]
Nhập file .json để tải lên, hoặc tải file Backup về máy.

Quản lý Hệ thống Tags

Tất cả các thẻ (Tags) đang được sử dụng. Bất kỳ thay đổi (đổi tên, xóa) ở đây sẽ được áp dụng trực tiếp lên toàn bộ Meme và Từ lóng đang chứa Tag đó.

Tự động phát Video
